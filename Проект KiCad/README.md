# Простейший блок питания (AC → DC)

## Что делает эта схема
Преобразует переменное напряжение (AC) с винтовых клемм J1 в постоянное (DC) на выходе J2. Светодиод D5 показывает наличие питания.

## Как работает
1. Диод D1 (1N4007) — однополупериодный выпрямитель
2. Конденсатор C1 — сглаживает пульсации
3. Резистор R1 + светодиод D5 — индикация
4. Резистор R2 — нагрузка (токоограничение)

## Компоненты
- J1, J2 — винтовые клеммы (вход/выход)
- D1 — 1N4007 (выпрямительный диод)
- C1 — электролитический конденсатор (фильтр)
- D5 — светодиод (индикатор)
- R1, R2 — резисторы
- H1-H4 — монтажные отверстия

## Для чего нужна
Может использоваться как простейший источник питания для других твоих проектов (вольтметра, платы на ATmega328P).
